# restaurant-dx-project
# 飲食店向けDXプロジェクト: カスタムAIマニュアルアプリ (MVP版)

## プロジェクトの目的
飲食店の現場（特に焼肉店など）における、新人スタッフの教育コスト削減と業務の属人化解消。
「理想」ではなく「現場で回る」ことを最優先とし、スタッフが自己解決できる環境を構築する。

## MVP（初期プロダクト）の概要
LINEのようにシンプルで直感的なUIを持つ、チャット型マニュアルアプリケーション。

## コア機能と技術要件
1. **データソースの完全な制御（NotebookLM方式）**
   - データベース（DB）は使用せず、店舗管理者が作成した `manualData.js` のみを知識源とする。
   - 【厳守】AIは `.js` ファイル内の情報のみを基に回答を生成する。
   - 【厳守】マニュアルに記載がない質問には「マニュアルに記載がありません。店長に確認してください」と回答し、ハルシネーション（嘘の回答）を完全に防ぐ。

2. **超軽量・高速なアーキテクチャ**
   - 複雑なバックエンドは持たず、フロントエンド（UI）と `.js` ファイルの読み込みのみで完結させる。
   - 現場の忙しい環境下でも、一瞬で回答が返ってくるレスポンス速度を担保する。

3. **現場特化型UI**
   - 検索窓ではなく、馴染みのある「チャット形式」を採用。
   - 「レジの締め方は？」「キャベツの切り方は？」など、現場の口語的な質問に対して的確にヒットさせる。
